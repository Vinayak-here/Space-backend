# space-backend

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Scripts

- **bun dev** – Runs the application directly from TypeScript for local development.
- **bun build** – Bundles the TypeScript source into production-ready JavaScript in the `dist` folder.
- **bun start** – Starts the application using the compiled output from the `dist` directory.
